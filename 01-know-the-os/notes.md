# 🧠 Module 01: Know the OS

## 📌 Overview
Understanding the Operating System (OS) is the foundation of cybersecurity. Ethical hackers primarily work in Linux-based environments like Kali Linux, which provides powerful tools for penetration testing.

---

## 🧩 Key Concepts

### 🔹 What is an Operating System?
An OS is software that manages hardware resources and provides services for programs.
Examples:
- Windows
- Linux
- macOS

---

### 🔹 Why Linux for Hacking?
- Open-source and customizable
- Powerful command-line interface
- Pre-installed tools in Kali Linux
- Better control over system processes

---

### 🔹 Linux File System Structure

| Directory | Purpose |
|----------|--------|
| /root | Root user home |
| /home | User directories |
| /etc | Configuration files |
| /var | Logs and variable data |
| /bin | Essential binaries |

---

### 🔹 Permissions System

Linux uses:
- Read (r)
- Write (w)
- Execute (x)

Example:
-rwxr-xr--
