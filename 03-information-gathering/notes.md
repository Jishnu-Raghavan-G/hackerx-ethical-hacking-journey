# 🧠 Module 03: Information Gathering

## 📌 Overview
Information gathering (OSINT) is the first phase of ethical hacking where data about a target is collected without direct interaction.

---

## 🧩 Key Concepts

### 🔹 Passive vs Active Recon
- Passive: No direct interaction
- Active: Direct probing (detectable)

---

### 🔹 Types of Information
- Domain details
- IP addresses
- DNS records
- Open ports

---

## ⚙️ Tools Covered
- whois
- nslookup
- nmap
- theHarvester

---

## 🧪 Techniques Learned
- Domain analysis
- Network scanning
- OSINT collection

---

## 💡 Key Takeaways
- Reconnaissance is critical
- More info = better attack strategy
