# 🧠 Module 02: Hack the MAC Address

## 📌 Overview
A MAC (Media Access Control) address is a unique identifier assigned to network interfaces. Changing it helps maintain anonymity during network activities.

---

## 🧩 Key Concepts

### 🔹 What is a MAC Address?
- Unique hardware identifier
- Format: XX:XX:XX:XX:XX:XX

---

### 🔹 Why Change MAC Address?
- Avoid tracking
- Bypass network restrictions
- Enhance privacy

---

### 🔹 Types of MAC Changes
- Random MAC
- Specific MAC spoofing

---

## ⚙️ Tools Covered
- macchanger
- ifconfig / ip

---

## 🧪 Techniques Learned
- Disabling network interface
- Spoofing MAC address
- Verifying changes

---

## 💡 Key Takeaways
- MAC spoofing improves anonymity
- Always change MAC before scanning networks
