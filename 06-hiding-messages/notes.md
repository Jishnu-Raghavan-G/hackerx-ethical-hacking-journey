# 🧠 Module 06: Hiding Messages (Steganography)

## 📌 Overview
Steganography hides information inside files like images, audio, or videos.

---

## 🧩 Key Concepts

### 🔹 Steganography vs Encryption
- Encryption: hides content
- Steganography: hides existence

---

### 🔹 Methods
- Image embedding
- Audio embedding

---

## ⚙️ Tools Covered

---

## 🧪 Techniques Learned
- Embedding hidden data
- Extracting hidden data

## 💡 Key Takeaways

- Hard to detect if done properly
